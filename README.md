# 📊 Interactive Sales Dashboard

## 📌 Project Overview

This project is an interactive sales dashboard built using **Streamlit**, **Plotly**, and **Pandas**.  
The dashboard helps visualize and analyze auto sales data through interactive charts, KPI cards, and filters.

The goal of this project is to transform raw sales data into meaningful business insights that are easy to understand for non-technical users.

---

# 🚀 Features

✅ Interactive Dashboard  
✅ Sidebar Filters  
✅ KPI Summary Cards  
✅ Dynamic Charts  
✅ Real-Time Data Filtering  
✅ Clean and User-Friendly UI  

---

# 📂 Dataset Used

The project uses the following dataset:

- `Cleaned_Auto_Sales.csv`

The dataset contains information such as:

- Product Line
- Country
- Sales
- Quantity Ordered
- Deal Size
- Order Date
- Customer Details

# 📊 Dashboard Components

## KPI Cards

The dashboard displays important business metrics such as:

- Total Sales
- Average Sales
- Total Orders
- Total Customers

---

# 📈 Charts Included

## 1. Sales by Product Line
- Bar Chart
- Shows total sales for each product category.

## 2. Sales Distribution by Country
- Pie Chart
- Displays sales contribution from different countries.

## 3. Monthly Sales Trend
- Line Chart
- Visualizes monthly sales growth and trends.

## 4. Quantity Ordered vs Sales
- Scatter Plot
- Analyzes relationship between quantity ordered and total sales.

---

# 🎛️ Interactive Filters

Users can filter dashboard data dynamically using:

- Product Line
- Country
- Deal Size
- Year

All charts and KPIs update automatically based on selected filters.
